## Version Control Integrations

<Tabs>
  <Tab title="GitHub" icon="github">
    Sync documentation with GitHub repositories for version control and collaboration.
    
    <Steps>
      <Step title="Connect Repository" icon="link">
        Link your GitHub repository to a documentation workspace.
        
        <CodeGroup tabs="JavaScript,cURL">
          ````javascript
const connection = await connectGitHub({
  repositoryUrl: 'https://github.com/user/repo',
  branch: 'main',
  token: 'ghp_your_token_here'
});
````
          ````bash
curl -X POST /api/integrations/github \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "repository_url": "https://github.com/user/repo",
    "branch": "main",
    "token": "ghp_your_token_here"
  }'
````
        </Step>
      <Step title="Auto-sync" icon="refresh-cw">
        Enable automatic synchronization of documentation changes.
      </Step>
    </Steps>
    
    <Callout kind="tip">Use GitHub Actions to automate documentation updates on code changes.</Callout>
  </Tab>

## Webhooks

Create custom integrations using our webhook system.

<Callout kind="info">Webhooks allow you to build custom integrations with any service that supports HTTP endpoints.</Callout>

<Steps>
  <Step title="Create Webhook" icon="webhook">
    Set up webhook endpoints in your workspace settings.
    
    <CodeGroup tabs="JavaScript,Python">
      ````javascript
const webhook = await createWebhook({
  url: 'https://your-app.com/webhook',
  events: ['document.created', 'document.updated'],
  secret: 'your-webhook-secret'
});
````
      ````python
webhook = api.create_webhook(
    url='https://your-app.com/webhook',
    events=['document.created', 'document.updated'],
    secret='your-webhook-secret'
)
````
    </Step>
  <Step title="Handle Events" icon="code">
    Process incoming webhook events in your application.
    
    <Expandable title="Event Types" default-open="false">
      - document.created
      - document.updated
      - document.deleted
      - workspace.member_added
      - comment.created
    </Expandable>
  </Step>
  <Step title="Verify Signatures" icon="lock">
    Use webhook signatures to verify request authenticity.
  </Step>
</Steps>
