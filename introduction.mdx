## Key Features

<Tabs>
  <Tab title="Organization" icon="folder">
    Organize your documentation with hierarchical structures, tags, and search capabilities.
    
    <Callout kind="tip">Use folders to group related documents and tags for quick filtering.</Callout>
    
    <Steps>
      <Step title="Create a Folder" icon="folder">
        Navigate to your dashboard and click the "New Folder" button.
        
        <CodeGroup tabs="JavaScript,Python">
          ````javascript
const folder = await createFolder({
  name: 'API Documentation',
  parentId: null
});
````
          ````python
folder = api.create_folder(name="API Documentation", parent_id=None)
````
        </Step>
      <Step title="Add Documents" icon="file-text">
        Upload or create documents within your folders.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Collaboration" icon="users">
    Collaborate with team members in real-time, share drafts, and manage permissions.
    
    <Expandable title="Permission Levels" default-open="false">
      - **Viewer**: Read-only access to documents
      - **Editor**: Can edit and comment on documents
      - **Admin**: Full control including user management
    </Expandable>
    
    <ExpandableGroup>
      <Expandable title="Real-time Editing" default-open="true">
        Multiple users can edit the same document simultaneously with conflict resolution.
      </Expandable>
      <Expandable title="Version History" default-open="false">
        Track all changes with automatic versioning and rollback capabilities.
      </Expandable>
    </ExpandableGroup>
  </Tab>
</Tabs>
