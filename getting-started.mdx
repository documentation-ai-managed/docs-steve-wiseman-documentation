## Setting Up Your Account

Welcome to Steve Wiseman Documentation! This guide will walk you through creating your account, setting up your first documentation space, and importing existing content.

<Steps>
  <Step title="Create Account" icon="user">
    Visit our website and click "Sign Up" to create your account.
    
    <CodeGroup tabs="JavaScript,cURL">
      ````javascript
const response = await fetch('/api/auth/signup', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'user@example.com',
    password: 'securepassword'
  })
});
````
      ````bash
curl -X POST /api/auth/signup \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","password":"securepassword"}'
````
    </Step>
  <Step title="Verify Email" icon="mail">
    Check your email for a verification link and click it to activate your account.
  </Step>
  <Step title="Create Workspace" icon="folder">
    Once logged in, create your first workspace for organizing documentation.
    
    <Tabs>
      <Tab title="Web Interface" icon="monitor">
        Click "New Workspace" in the dashboard and enter a name like "Project Docs".
      </Tab>
      <Tab title="API" icon="code">
        Use our API to create workspaces programmatically.
        
        ````javascript
const workspace = await createWorkspace({
  name: 'Project Docs',
  description: 'Documentation for our main project'
});
````
      </Tab>
    </Tabs>
  </Step>
</Steps>

## Creating Your First Document

<Steps>
  <Step title="Navigate to Workspace" icon="folder">
    Click on your workspace from the dashboard.
  </Step>
  <Step title="New Document" icon="file-text">
    Click "New Document" and choose a template or start blank.
    
    <CodeGroup tabs="JavaScript,Python">
      ````javascript
const doc = await createDocument({
  title: 'API Reference',
  content: '# API Reference\n\nThis document describes our API endpoints.',
  workspaceId: workspace.id
});
````
      ````python
doc = api.create_document(
    title="API Reference",
    content="# API Reference\n\nThis document describes our API endpoints.",
    workspace_id=workspace.id
)
````
    </Step>
  <Step title="Edit and Save" icon="edit">
    Use our rich text editor to add content, then save your changes.
  </Step>
</Steps>
